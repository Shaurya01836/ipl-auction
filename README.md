## 🚀 Elevate Your Auction Experience
The **IPL Mega Auction 2026 Simulation Platform** is a high-performance, real-time SaaS solution designed to replicate the intensity and professional aesthetic of the official IPL auction. Built with a broadcast-first philosophy, it combines low-latency bidding technology with advanced AI-driven analytics.

---

## ✨ Key Features

### 🏢 **Enterprise-Grade Bidding Engine**
- **Real-Time Synchronization**: Powered by Firebase Realtime Database for sub-second latency across all participants.
- **Atomic Transactions**: Ensured data integrity for every bid, preventing overlaps and race conditions.
- **Smart Countdown**: Universal time synchronization ensures every manager sees the same clock, down to the millisecond.

### 🧠 **Gemini AI Analytics Engine**
- **Live Squad Valuation**: Dynamic scoring based on batting strike rates, bowling economy, and team balance.
- **Predictive Points Table**: Automatically ranks franchises based on draft quality and tactical recruitment.
- **Behavioral Safeguards**: Integrated disqualification logic (18-player minimum) to enforce professional squad building.

### 📺 **Broadcast-Quality UI/UX**
- **High-Fidelity Branding**: Official IPL franchise logos, colors, and premium glassmorphism design.
- **Dynamic Overlays**: Animated celebration screens for successful buys and automated "Unsold" notifications.
- **Infinite Logo Marquee**: A stunning, branded landing experience for immediate user engagement.
- **Universal Typography**: Utilizing the **Inter** font family for a sleek, modern, and readable interface.

### 📊 **Advanced Roster Management**
- **Set-Wise Grouping**: Organize upcoming players by official auction segments with centered headers.
- **Historical Tracking**: Comprehensive "History" tab with chronological sorting and acquisition price visibility.
- **Export Capabilities**: Professional squad summaries and leaderboard views for post-auction analysis.

## 🛠 Tech Stack

- **Frontend**: React 18, Vite (Ultra-fast HMR)
- **Styling**: Tailwind CSS v4, Framer Motion (Broadcast Animations)
- **Backend**: Firebase (Auth, Firestore, Realtime Database)
- **Icons**: Lucide React
- **Typography**: Inter (via Google Fonts)

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn
- Firebase Project setup

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shaurya01836/ipl-auction.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env` file in the root directory and add your Firebase credentials.

4. **Launch Development Server**:
   ```bash
   npm run dev
   ```
